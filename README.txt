The folders in this directory contain the following:

content ->	

	Images, html, javascript, and css files that make up the front end of the application.
	Note: the folder named 'light' is from a template called 'Mist Multipurpose HTML Template'
	and was purchased here - http://themeforest.net/item/mist-multipurpose-html5-responsive-template/9997523

extensions ->

	Third-party application extensions that are either currentlly being used, or can be used
	in the future. Right now it's all ADO.NET database providers.


source ->

	The source code of the project.
